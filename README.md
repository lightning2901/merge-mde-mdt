# mde-mdt
Proyecto final de clase de vis 3D. En vez de hacer el modelo digital del terreno y de elevaciones en Qgis o Arcgis, se realiza con Python.

## Observaciones
1.- Para ejecutar el programa úsese un cuaderno como Jupyter notebook
2.- Úsese un entorno virtual
3.- En el caso del Modelo Digital del Terreno: Es necesario cambiar en el código el valor de Azimuth del sol (Si no estás muy familiarizado, va de 0 a 180 grados)

## Cómo funciona
Lo abres en jupyter notebook o colab (por dar unos ejemplos), requieres de una imagen ráster de entrada.
